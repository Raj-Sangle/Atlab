# ATLAB K-12 Education Website

A modern, responsive website for ATLAB's K-12 education solutions, featuring STEM, robotics, AI, and immersive learning technologies.

## Features

- Responsive design for all devices
- Interactive tabs for different education categories
- Animated counters and statistics
- Image carousels and sliders
- Modern CSS animations with AOS library
- Sticky navigation header

## Technologies Used

- HTML5
- CSS3 with modern features
- Vanilla JavaScript
- AOS (Animate On Scroll) library
- Responsive design principles
